模块分离框架 
建议项目目录
本招聘数据智能系统着重研究了招聘市场中信息获取、处理和利用等问题，并提出了基于八爪鱼采集器数据爬取、Pandas清洗分析和可视化、机器学习及深度学习算法（包括不限于Kmeans聚类算法、神经网络分类算法）运用于NLP招聘数据深入分析、大模型API的智能客服及智能分析的招聘数据解决方案。系统主要功能模块包括：（1）数据爬取模块：使用八爪鱼采集器对诸如BOSS直聘网站、智联招聘爬取不同招聘岗位不少于15000条招聘数据；（2）数据预处理模块：使用Pandas对爬取的数据进行数据清理、数据变换、数据规约等处理；（3）数据可视化模块：薪资分析：通过可视化图表展示不同岗位的月平均薪资及年终奖的分布；企业分析：直观呈现企业地域分布、类型及规模特征；岗位需求：通过词云技术提炼岗位技能要求与福利待遇关键词；学历要求：统计学历与工作经验的市场需求分布；企业融资：分析行业技术热点与融资阶段分布；（4）机器学习模块：对招聘详情运用NLP进行文本表示，并使用Kmeans聚类算法与神经网络分类算法进行机器学习与自动识别；（5）AI智能交互模块，基于大模型API技术提供定制化求职建议；（6）基于Streamlit实现招聘数据智能系统，包括核心模块：用户登录注册、数据可视化（若干简单统计、词云图、Kmean聚类）、智能客服等。
这个项目涉及数据爬取、数据预处理、模型推理、app应用等，应尽可能采用模块分离开发的策略，比如以下设计了一个模块解耦、结构清晰的项目目录。目录将各个模块合理地划分到了数据层、算法层、交互层和展现层中。

Streamlit介绍
Streamlit 是一个用于快速创建数据科学和机器学习 Web 应用程序的开源 Python 框架。它让数据科学家和工程师能够轻松地将他们的 Python 脚本转换为交互式 Web 应用，而无需前端开发经验。主要特点如下：
简单易用：只需几行代码即可创建功能齐全的 Web 应用
纯 Python：不需要学习 HTML、CSS 或 JavaScript
快速开发：即时保存和查看更改，开发效率高
丰富的组件：内置多种数据可视化、交互式控件和布局选项
与主流数据科学与人工智能生态系统兼容：支持 Pandas、Matplotlib、Plotly、PyTorch 等

常见错误及解决方法
安装pip install volcengine-python-sdk[ark]报错，如果遇到提示说目录不存在,可能因为临时路径长度太长了。
解决方案：1. 按下 Win+R ，输入 regedit 打开注册表编辑器。2. 设置 \HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem 路径下的变量 LongPathsEnabled 为 1 即可。
解决方案：2.源码安装：
https://pypi.tuna.tsinghua.edu.cn/packages/c5/98/f0d0eaecdf89b0a89a1b334eab53367c2388e34a04e14c367e8003587d9b/volcengine-python-sdk-1.0.121.tar.gz 
运行：
python setup.py install --user

TRAE或VS CODE终端运行中切换conda的python虚拟器报错

大模型API_KEY设置完代码不能正常使用，请重启TRAE或者VS CODE。
